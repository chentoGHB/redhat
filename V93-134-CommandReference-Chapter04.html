📦 tar Archive Commands
Create, list, and extract tar archives for secure data packaging

#tar -czf backup.tar.gz /etc

#tar -cjf backup.tar.bz2 /var/log

$tar -cJf backup.tar.xz /home/user
Create xz-compressed archive (best compression)
-J enables xz compression (.tar.xz)
Click to copy
$tar -tf archive.tar.gz
List contents of tar archive
-t (list), works with compressed and uncompressed archives
Click to copy
$tar -tvf archive.tar.gz
List archive contents with detailed information
-v (verbose) adds permissions, ownership, timestamps
Click to copy
$tar -xf archive.tar
Extract tar archive to current directory
-x (extract), auto-detects compression
Click to copy
$tar -xzf backup.tar.gz -C /tmp
Extract archive to specific directory
-C changes directory before extraction
Click to copy
#tar -xvpf archive.tar --xattrs --selinux
Extract with permissions, extended attributes, and SELinux contexts
-p preserves permissions, --xattrs and --selinux preserve security contexts

🔄 rsync Synchronization
Efficient file synchronization and incremental backup tool
Click to copy
$rsync -av /source/ /destination/
Basic local directory synchronization
-a (archive mode), -v (verbose)
Click to copy
$rsync -av /local/dir user@remote:/remote/dir
Sync local directory to remote system
Remote format: [user@]host:/path
Click to copy
$rsync -av user@remote:/remote/dir /local/backup/
Sync remote directory to local system
Pull remote data for backup
Click to copy
$rsync -avz /large/dir/ user@remote:/backup/
Sync with compression for slow networks
-z compresses data during transfer
Click to copy
$rsync -n -av /source/ /dest/
Dry run - show what would be synchronized
-n performs simulation without changes
Click to copy
$rsync -av --delete /source/ /dest/
Sync and delete files not in source
--delete removes extra files in destination
Click to copy
#rsync -avAX /source/ /dest/
Sync with ACLs and SELinux contexts
-A (ACLs), -X (SELinux contexts) for security
Click to copy
$rsync -av --progress /large/files/ /backup/
Show transfer progress for large files
--progress displays transfer statistics
Click to copy
$rsync -av --exclude="*.tmp" /source/ /dest/
Exclude files matching pattern
--exclude filters out unwanted files

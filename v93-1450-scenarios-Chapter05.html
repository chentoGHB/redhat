‚û°Ô∏è Redirection & Output Control

Control command output flow using redirection operators to save results to files or append to existing content.

Redirection Operators Output Control
$ ls -l > listing.txt
$ date >> logfile.txt
$ command 2> error.log
üéØ Redirection Operators Explained:
> (Redirect): Send output to file, overwrite existing content
>> (Append): Add output to end of file, preserve existing content
2> (Error Redirect): Redirect error messages to file
&> (All Output): Redirect both stdout and stderr to file
< (Input Redirect): Use file as input for command

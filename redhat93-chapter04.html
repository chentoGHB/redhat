Reading Manual Pages

The man Command

Manual pages (man pages) are the primary source of documentation for Linux commands. They provide detailed information about command syntax, options, and examples.

man command
Display the manual page for a specific command
$ man ls
LS(1) User Commands LS(1)

NAME
ls - list directory contents

SYNOPSIS
ls [OPTION]... [FILE]...

DESCRIPTION
List information about the FILEs (the current directory by default).
Sort entries alphabetically if none of -cftuvSUX nor --sort is specified.
man section command
Display manual page from specific section
$ man 5 passwd
PASSWD(5) File Formats Manual PASSWD(5)

NAME
passwd - password file

DESCRIPTION
The /etc/passwd file contains one line for each user account

Manual Page Sections

1: User commands
2: System calls
3: Library functions
4: Device files
5: File formats and configuration files
6: Games
7: Miscellaneous
8: System administration commands

whatis command
Display brief description of a command
$ whatis ls
ls (1) - list directory contents
man -f command
Same as whatis - show brief command description
$ man -f cp
cp (1) - copy files or directories

Navigate and Search man Pages

Navigation Keys in man Pages

When viewing a man page, you can use various keys to navigate through the content:

Navigation Commands

Space or Page Down - Move forward one page
b or Page Up - Move backward one page
Enter or Down Arrow - Move forward one line
Up Arrow - Move backward one line
g - Go to beginning of document
G - Go to end of document
q - Quit the man page

Searching Within man Pages

/search_term
Search forward for a term within the man page
/recursive
[Highlights and jumps to first occurrence of "recursive"]
?search_term
Search backward for a term within the man page
?option
[Highlights and jumps to previous occurrence of "option"]
n
Jump to next occurrence of search term
N
Jump to previous occurrence of search term
$ man ls
[In man page, type:]
/long
[Press 'n' to find next occurrence]
[Press 'N' to find previous occurrence]
[Press 'q' to quit]

Getting Help Within man

h
Display help for man page navigation (while in man page)
[While viewing a man page, press 'h']
SUMMARY OF LESS COMMANDS

Commands marked with * may be preceded by a number, N.
Notes in parentheses indicate the behavior if N is given.

h H Display this help.
q :q Q :Q ZZ Exit.

Search for man Pages by Keyword

The apropos Command

apropos keyword
Search manual page names and descriptions for keyword
$ apropos copy
cp (1) - copy files or directories
cpio (1) - copy files to and from archives
dd (1) - convert and copy a file
install (1) - copy files and set attributes
rsync (1) - a fast, versatile, remote (and local) file-copying tool
man -k keyword
Same as apropos - search by keyword
$ man -k network
ifconfig (8) - configure a network interface
netstat (8) - Print network connections, routing tables
ping (8) - send ICMP ECHO_REQUEST to network hosts
ssh (1) - OpenSSH SSH client (remote login program)
apropos -e keyword
Search for exact keyword matches
$ apropos -e file
file (1) - determine file type
file (1p) - determine file type
apropos "phrase with spaces"
Search for phrases containing spaces
$ apropos "file system"
df (1) - report file system disk space usage
fsck (8) - file system check
mount (8) - mount a file system

Using Multiple Keywords

apropos keyword1 keyword2
Search for commands containing any of the keywords
$ apropos user group
chgrp (1) - change group ownership
groups (1) - print the groups a user is in
id (1) - print real and effective user and group IDs
useradd (8) - add a user account
usermod (8) - modify a user account
apropos -a keyword1 keyword2
Search for commands containing ALL keywords (AND operation)
$ apropos -a user add
useradd (8) - add a user account

Additional Help Commands

info command
Display info documents (more detailed than man pages)
$ info ls
File: coreutils.info, Node: ls invocation, Next: dir invocation

10.1 'ls': List directory contents
=====================================

The 'ls' program lists information about files (of any type, including
directories). Options and file arguments can be intermixed arbitrarily
command --help
Display brief help for most commands
$ ls --help
Usage: ls [OPTION]... [FILE]...
List information about the FILEs (the current directory by default).
Sort entries alphabetically if none of -cftuvSUX nor --sort is specified.

Mandatory arguments to long options are mandatory for short options too.
-a, --all do not ignore entries starting with .
mandb
Update the manual page database (run as root)
$ sudo mandb
Processing manual pages under /usr/share/man...
Updating index cache for path `/usr/share/man/man1'. Wait...
Checking for stray cats under /usr/share/man...
Checking for stray cats under /var/cache/man...

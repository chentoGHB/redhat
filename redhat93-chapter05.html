Standard Input, Standard Output, and Standard Error

Understanding Standard Streams

Linux uses three standard data streams for input and output operations:

stdin (0)
stdout (1)
stderr (2)
  • stdin (0): Standard input - keyboard input by default
  • stdout (1): Standard output - screen display by default
  • stderr (2): Standard error - error messages to screen by default

Viewing Standard Streams in Action

echo "Hello World"
Send text to standard output
$ echo "Hello World"
Hello World
cat
Read from standard input and write to standard output
$ cat
This is a test
This is a test
[Ctrl+D to exit]
ls nonexistent
Demonstrate standard error output
$ ls nonexistent
ls: cannot access 'nonexistent': No such file or directory

Redirect Output to a File

Output Redirection

command > file
Redirect stdout to a file (overwrites existing content)
$ echo "Hello World" > greeting.txt
$ cat greeting.txt
Hello World
command >> file
Append stdout to a file (preserves existing content)
$ echo "Second line" >> greeting.txt
$ cat greeting.txt
Hello World
Second line
command 2> file
Redirect stderr to a file
$ ls nonexistent 2> error.log
$ cat error.log
ls: cannot access 'nonexistent': No such file or directory
command &> file
Redirect both stdout and stderr to a file
$ ls existing nonexistent &> combined.log
$ cat combined.log
existing
ls: cannot access 'nonexistent': No such file or directory

Input Redirection

command < file
Use file content as standard input
$ wc -l < greeting.txt
2
command << EOF
Here document - input until EOF marker
$ cat << EOF > multi.txt
> Line 1
> Line 2
> Line 3
> EOF
$ cat multi.txt
Line 1
Line 2
Line 3

Construct Pipelines

Basic Pipes

command1 | command2
Pipe stdout of command1 to stdin of command2
$ ls -l | head -5
total 16
-rw-r--r-- 1 student student 25 Jul 11 10:30 combined.log
-rw-r--r-- 1 student student 67 Jul 11 10:30 error.log
-rw-r--r-- 1 student student 25 Jul 11 10:30 greeting.txt
-rw-r--r-- 1 student student 21 Jul 11 10:30 multi.txt
ps aux | grep process
Find specific processes
$ ps aux | grep bash
student 1234 0.0 0.1 21312 3584 pts/0 S+ 10:30 0:00 bash
student 5678 0.0 0.0 12345 1024 pts/1 S+ 10:31 0:00 grep bash

Complex Pipelines

cat file | sort | uniq
Chain multiple commands together
$ cat names.txt | sort | uniq
Alice
Bob
Charlie
David
history | tail -10 | head -5
Get lines 6-10 from the end of history
$ history | tail -10 | head -5
491 ls -l
492 cat greeting.txt
493 echo "test"
494 pwd
495 cd Documents

Useful Pipeline Commands

command | tee file
Write output to both file and stdout
$ ls -l | tee listing.txt
total 16
-rw-r--r-- 1 student student 25 Jul 11 10:30 greeting.txt
[Output also saved to listing.txt]
command | xargs
Convert input to command arguments
$ echo "file1.txt file2.txt" | xargs touch
$ ls file*.txt
file1.txt file2.txt

Edit Files with Vim

Starting Vim

vim filename
Open file in Vim editor
$ vim myfile.txt
[Vim editor opens with myfile.txt]

Vim Modes

Normal Mode: Default mode for navigation and commands
Insert Mode: For typing and editing text
Visual Mode: For selecting text
Command Mode: For executing Vim commands

Basic Vim Commands

i
Enter insert mode at cursor position
a
Enter insert mode after cursor
o
Open new line below and enter insert mode
Esc
Return to normal mode from any other mode

Navigation in Vim

Movement Commands

h - Move left
j - Move down
k - Move up
l - Move right
w - Move to beginning of next word
b - Move to beginning of previous word
0 - Move to beginning of line
$ - Move to end of line

Saving and Exiting

:w
Save the file
:q
Quit Vim (only if no changes)
:wq
Save and quit
:q!
Quit without saving changes
[In Vim, type:]
i
Hello, this is my first Vim file!
[Press Esc]
:wq
[File saved and Vim exits]

Rearrange Existing Text in Vim

Delete Commands

x
Delete character under cursor
dw
Delete word from cursor position
dd
Delete entire line
d$
Delete from cursor to end of line

Copy and Paste

yy
Copy (yank) entire line
yw
Copy word from cursor position
p
Paste after cursor
P
Paste before cursor

Undo and Redo

u
Undo last change
Ctrl+r
Redo last undone change
[In Vim Normal Mode:]
yy [Copy current line]
p [Paste line below]
dd [Delete current line]
u [Undo delete]
3dd [Delete 3 lines]

Search and Replace

/pattern
Search forward for pattern
?pattern
Search backward for pattern
n
Go to next search result
N
Go to previous search result
:%s/old/new/g
Replace all occurrences of 'old' with 'new'
[In Vim Command Mode:]
/hello
[Searches for "hello"]
n
[Goes to next occurrence]
:%s/hello/hi/g
[Replaces all "hello" with "hi"]

v

y

Change the Shell Environment

Environment Variables

env
Display all environment variables
$ env
HOME=/home/student
PATH=/usr/local/bin:/usr/bin:/bin
USER=student
SHELL=/bin/bash
PWD=/home/student
echo $VARIABLE
Display value of specific environment variable
$ echo $HOME
/home/student
export VARIABLE=value
Set and export environment variable
$ export EDITOR=vim
$ echo $EDITOR
vim

PATH Variable

echo $PATH
Display current PATH variable
$ echo $PATH
/usr/local/bin:/usr/bin:/bin:/usr/sbin:/sbin
export PATH=$PATH:/new/path
Add directory to PATH
$ export PATH=$PATH:/home/student/bin
$ echo $PATH
/usr/local/bin:/usr/bin:/bin:/usr/sbin:/sbin:/home/student/bin

Configure Bash with Shell Variables

Local Variables

VARIABLE=value
Set local shell variable (not exported)
$ NAME="John Doe"
$ NAME="John Doe"
$ echo $NAME
John Doe
set
Display all shell variables and functions
$ set | head -5
BASH=/bin/bash
BASHOPTS=checkwinsize:cmdhist:complete_fullquote
BASHPID=12345
BASH_ALIASES=()
BASH_ARGC=()

Shell Configuration Files

~/.bashrc
Personal bash configuration file (executed for interactive shells)
~/.bash_profile
Personal bash profile (executed for login shells)
source ~/.bashrc
Reload bash configuration without logging out
$ echo 'export EDITOR=vim' >> ~/.bashrc
$ source ~/.bashrc
$ echo $EDITOR
vim

Command Aliases

alias name='command'
Create command alias
$ alias ll='ls -l'
$ ll
total 16
-rw-r--r-- 1 student student 25 Jul 11 10:30 greeting.txt
-rw-r--r-- 1 student student 21 Jul 11 10:30 multi.txt
alias
Display all current aliases
$ alias
alias ll='ls -l'
alias la='ls -la'
alias grep='grep --color=auto'
unalias name
Remove an alias
$ unalias ll
$ ll
bash: ll: command not found

History Configuration

HISTSIZE=1000
Set number of commands to keep in memory
HISTFILESIZE=2000
Set number of commands to keep in history file
$ export HISTSIZE=500
$ export HISTFILESIZE=1000
$ echo $HISTSIZE
500

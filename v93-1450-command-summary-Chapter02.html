üìÑ File Content and Information
‚ñº
file filename
Determine file type and characteristics
[student@workstation ~]$ file zcat
zcat: POSIX shell script, ASCII text executable
wc filename
Word count - display lines, words, and characters
[student@workstation ~]$ wc zcat
91 385 2796 zcat
head filename
Display first 10 lines of a file
[student@workstation ~]$ head zcat
#!/bin/sh
# zcat -- gzip decompression
tail filename
Display last 10 lines of a file
[student@workstation ~]$ tail zcat
exec gzip -cd "$@"
head -n 20 filename
Display first 20 lines of a file
[student@workstation ~]$ head -n 20 zcat
tail -n 20 filename
Display last 20 lines of a file
[student@workstation ~]$ tail -n 20 zcat
cat filename
Display entire file contents
[student@workstation ~]$ cat /etc/hostname
workstation.example.com
less filename
View file contents page by page (interactive)
[student@workstation ~]$ less /var/log/messages

üîç Search & Find Commands
‚ñº
man -k (apropos)
Search manual page names and descriptions by keyword
[student@workstation ~]$ man -k passwd
passwd (1) - update user's authentication tokens
passwd (5) - password file
[student@workstation ~]$ man -k network
Lists all network-related commands
üîç Research Tips
  • Use man -k security to find security tools
  • Search for man -k firewall or man -k encryption
  • Essential for discovering new security commands
man -K
Full-text search within manual pages (more comprehensive but slower)
[student@workstation ~]$ man -K firewall
--Man-- next: iptables(8) [ view (return) | skip (Ctrl-D) | quit (Ctrl-C) ]
‚ö° Search Strategy
  • Use when keyword searches don't find what you need
  • Press Enter to view, Ctrl-D to skip, Ctrl-C to quit
  • Excellent for finding security concepts in documentation
whereis
Locate binary, source, and manual page files for commands
[student@workstation ~]$ whereis passwd
passwd: /usr/bin/passwd /etc/passwd /usr/share/man/man1/passwd.1.gz
[student@workstation ~]$ whereis nmap
Shows location of nmap binary and manual
üóÇÔ∏è System Investigation
  • Verify security tool installations and locations
  • Find configuration files associated with commands
  • Useful for forensic investigation of system binaries

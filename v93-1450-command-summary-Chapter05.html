Output Redirection
Redirect standard output and error streams to files for logging and analysis

‚úèÔ∏è Text Editing with Vim
‚ñº
vim / vi
Powerful terminal text editor essential for system administration and configuration
Command Mode
Navigate, cut, paste, delete. Default mode when opening vim.
Insert Mode (i)
Type text content. Press Esc to return to command mode.
Visual Mode (v)
Select text. Shift+V for lines, Ctrl+V for blocks.
Extended Mode (:)
Commands like :w (save), :q (quit), :wq (save & quit).
[admin@security-server ~]$ vim /etc/ssh/sshd_config
Edit SSH daemon configuration
[admin@security-server ~]$ vim ~/.bashrc
Customize shell environment
‚öôÔ∏è Essential Commands
  • :w - Save file, :q - Quit, :wq - Save and quit
  • :q! - Quit without saving changes
  • u - Undo, x - Delete character
  • dd - Delete line, yy - Copy line, p - Paste
vimtutor
Interactive vim tutorial for learning essential editing skills
[student@workstation ~]$ vimtutor
Launch interactive vim tutorial
üìö Learning Strategy
  • Essential for security professionals who manage servers
  • Learn basic navigation: h, j, k, l (left, down, up, right)
  • Practice with configuration files in safe environment
  • Master vim to edit files over SSH connections

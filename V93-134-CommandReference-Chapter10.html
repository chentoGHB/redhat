GRUB Boot Editor Procedure:
Steps to edit kernel command line at boot
1. Boot/reboot system
2. Press any key (except Enter) to interrupt countdown
3. Highlight kernel entry with cursor keys
4. Press 'e' to edit entry
5. Navigate to line starting with 'linux'
6. Press End, append parameters
7. Press Ctrl+X to boot with changes

Complete Password Recovery Steps:
Complete rd.break password recovery procedure
1. Boot to rescue kernel entry
2. Edit kernel line, append 'rd.break'
3. Press Ctrl+X to boot
4. mount -o remount,rw /sysroot
5. chroot /sysroot
6. passwd root
7. touch /.autorelabel
8. exit (twice)

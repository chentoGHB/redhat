🎛️ Tuning Profiles Management
Manage system performance profiles for different workloads
Click to copy
$tuned-adm list
List all available tuning profiles
- balanced - General non-specialized tuned profile
- desktop - Optimize for the desktop use-case
- latency-performance - Optimize for deterministic performance
- network-latency - Low latency network performance
- powersave - Optimize for low power consumption
- throughput-performance - Broadly applicable tuning
- virtual-guest - Optimize for running inside a virtual guest

$tuned-adm active

📊 Process Priority Control
Manage process scheduling priority with nice values
Click to copy
$nice command
Start command with default nice value (+10)
Lower priority than current shell
Click to copy
$nice -n 15 cpu_intensive_task
Start command with specific nice value
Higher nice value = lower priority (range: -20 to +19)
Click to copy
#nice -n -5 critical_security_scan
Start high-priority security process
Negative nice values require root privileges
Click to copy
$renice -n 19 1234
Change nice value of existing process by PID
1234 (process ID) old priority 0, new priority 19
Click to copy
#renice -n -10 $(pgrep sshd)
Increase priority of SSH daemon processes
Critical for secure remote access
Click to copy
#renice -n 5 -u student
Change nice value for all processes of a user
Useful for limiting user resource consumption
Click to copy
#renice -n 10 -g apache
Change nice value for process group
Control web server process priority
